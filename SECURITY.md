# Security Policy

## Supported Versions

We actively support and patch security issues on the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do NOT file a public issue. Instead, report it privately.

Please contact the repository owner directly. Include:
* A detailed description of the vulnerability.
* Steps to reproduce the issue (including proof of concept scripts if possible).
* Potential impacts if exploited.

We aim to acknowledge receipt of your report within 48 hours, and provide a resolution plan within 7 days.
