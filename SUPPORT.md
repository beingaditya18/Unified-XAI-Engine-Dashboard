# Support Policy

Thank you for utilizing the Unified XAI Engine! Please use the following channels to resolve configuration issues or ask questions:

## 🔍 Questions & Troubleshooting

If you have general usage questions or need help setting up the pipeline in your environment:
* Check out the [README](README.md) installation guidelines.
* Review the existing issues list to see if a similar problem has already been addressed.
* Open a new discussion thread in the GitHub Discussions board.

## 🐛 Bug Reports

If you've identified a bug or error in the source code:
* Search current issues to see if the bug has been filed.
* Open a bug report issue using our [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.yml).

## 💡 Feature Proposals

If you have a feature idea or architectural improvement:
* File a request using the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.yml) to outline your suggestions.
